Extract Data from Passport with just one click


About Project

This is a Script developed for extracting user details from the image of their passport. This is designed for indian passports.

Extraction

This is a project designed for extracting data from passport images. It is done by using "OpenCV-Python" for image processing and Pytesseract-OCR(Optical Character Recognition) for text extraction.

For the extraction of data from passport image, at first it undergoes through a series of image processing to ensure that the image is in correct for and it is in a state from where wee can extract data.

Step1-Converting Image to array

The first step of image is to convert the image into an array. This is an essential process for image processing.

Step2-Convert to gray scale

Next we will convert the image to grayScale to get maximum details.

Step3-Sharpen the image

The image is then sharpened to get crisp the details in the image. This help in better border recognisions and improve recognitions.

Step4-Denoise the image

The image will undergo denoising. This will remove the unneccessary details from the image.

Step5-Text Extraction

Text Extraction is done by using Pytesseract-OCR module python.

Step6-Filter by Confidence

Only texts with a confidence level greater than 60 are considered. The confidence value is used to filter out less reliable detections.

Step7-Using Regex Patterns for Information Extraction

When extracting with Pytesseract-OCR we get it as lines of data. Convert it into a list for easy processing. The informations like Gender, Passport Number, Date Patterns, Nationality, Country Code... etc is extracteed by using RedEx patterns.

Step8-To Json

This is then changed to json format for easy handling accross diffrent languages and frameworks.
